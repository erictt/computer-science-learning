Proof the Chain Rule

  • Recall If y = f(x) and x changes from a to a + ∆x, we define the increment of y as $$\Delta y = f(a+\Delta x)-f(a)$$. According to the definition of a derivative, we have $$\lim_{\Delta x \to o}\frac{\Delta y}{\Delta x}=f’(a)$$ . So if we denote by ϵ\epsilonϵ the difference between the difference quotient and the derivative, we obtain $$\lim_{\Delta x \to 0}\epsilon = \lim_{\Delta x \to 0}(\frac{\Delta y}{\Delta x}-f’(a)) = f’(a)-f’(a) = 0$$. But $$\epsilon = \frac{\Delta y}{\Delta x}-f’(a)\ \Rightarrow \Delta y = f’(a)\Delta x + \epsilon \Delta x$$. If we define ϵ\epsilonϵ to be 0 when ∆x = 0, then ϵ\epsilonϵ become a continuous function of ∆x. Thus, for a differentiable function f, we can write $$\Delta y = f’(a)\Delta x + \epsilon \Delta x \text{ where } \epsilon \to 0\ as\ \Delta x \to 0$$ and ϵ\epsilonϵ is a continuous function of ∆x. This property of differentiable functions is what enables us to prove the Chain Rule.
  • Now to Prove: Suppose u=g(x) is differentiable at a and y=f(u) is differentiable at b=g(a), If ∆x is an increment in x and ∆u and ∆y are corresponding increments in u and y, then we can use last equation to write $$\Delta u = g’(a)\Delta x + \epsilon_1\Delta x = (g’(a) + \epsilon_1)\Delta x$$ where ϵ1→0\epsilon_1 \to 0ϵ1​→0 as Δx→0\Delta x \to 0Δx→0. Similarly $$\Delta y = f’(b)\Delta u + \epsilon_2\Delta u = (f’(b) + \epsilon_2)\Delta u$$ where ϵ2→0\epsilon_2 \to 0ϵ2​→0 as Δx→0\Delta x \to 0Δx→0. If we now substitute the expression for ∆u, we get $$\Delta y = [f’(b) + \epsilon_2][g’(a) + \epsilon_1]\Delta x$$, so $$\frac{\Delta y}{\Delta x} = [f’(b) + \epsilon_2][g’(a) + \epsilon_1]$$. As Δx→0\Delta x \to 0Δx→0. So both ϵ2→0\epsilon_2 \to 0ϵ2​→0 and ϵ1→0\epsilon_1 \to 0ϵ1​→0 as Δx→0\Delta x \to 0Δx→0. Therefore

    • dydx=lim⁡Δx→0ΔyΔx=lim⁡Δx→0[f′(b)+ϵ2][g′(a)+ϵ1]=f′(b)g′(a)=f′(g(a))g′(a)\begin{aligned} \frac{dy}{dx} &= \lim_{\Delta x \to 0}\frac{\Delta y}{\Delta x} = \lim_{\Delta x \to 0}[f'(b) + \epsilon_2][g'(a) + \epsilon_1] \\ &= f'(b)g'(a) = f'(g(a))g'(a) \end{aligned}dxdy​​=Δx→0lim​ΔxΔy​=Δx→0lim​[f′(b)+ϵ2​][g′(a)+ϵ1​]=f′(b)g′(a)=f′(g(a))g′(a)​
