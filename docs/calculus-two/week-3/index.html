Integral Test

Theorem

  • Suppose f is a continuous, positive, decreasing function on [1,∞)[1, \infty)[1,∞), and let an=f(n)a_n = f(n)an​=f(n). Then the series ∑n=1∞an\sum_{n=1}^{\infty} a_n∑n=1∞​an​ is convergent if and only if the improper integral ∫1∞f(x)dx\int_{1}^{\infty} f(x) dx∫1∞​f(x)dx is convergent. In other words:
    • If ∫1∞f(x)dx\int_{1}^{\infty} f(x) dx∫1∞​f(x)dx is convergent, then ∑n=1∞an\sum_{n=1}^{\infty} a_n∑n=1∞​an​ is convergent.
    • If ∫1∞f(x)dx\int_{1}^{\infty} f(x) dx∫1∞​f(x)dx is divergent, then ∑n=1∞an\sum_{n=1}^{\infty} a_n∑n=1∞​an​ is divergent.
