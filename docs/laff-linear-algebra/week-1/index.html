Other Norms

  • A norm is a function, in our case of a vector in Rn\mathbb{R}^nRn, that maps every vector to a nonnegative real number. The simplest example is the absolute value of a real number: Given α∈R\alpha \in \mathbb{R}α∈R, the absolute value of α, often written as |α|, equals the magnitude of α:

    • ∣α∣={αif α≥0,−αotherwise.\lvert \alpha \rvert = \left\{ \begin{array}{rl} \alpha & \text{if } \alpha \ge 0,\\ -\alpha & \text{otherwise}. \end{array} \right.∣α∣={α−α​if α≥0,otherwise.​

  • Similarly, one can find functions, called norms, that measure the magnitude of vectors. One example is the (Euclidean) length of a vector, which we call the 2-norm: for x∈Rnx \in \mathbb{R}^nx∈Rn,

    • ∥x∥2=∑i=0n−1xi2\lVert x \rVert _2 = \sqrt{\sum_{i=0}^{n-1}x_i^2} ∥x∥2​=i=0∑n−1​xi2​​

  • Other norms:
    • 1-norm (also called taxi-cab norm): $$\lVert x \rVert 1 = \sqrt{\sum{i=0}^{n-1}|x_i|}$$.
    • For 1≤p≤∞1 \le p \le \infty1≤p≤∞, the p-norm: $$\lVert x \rVert p = \sqrt[p]{\sum{i=0}{n-1}|x_i|p}$$.

Summary of the Properties for Vector Operations

Vector Addition

  • Is commutative. That is, for all vectors x,y∈Rn,x+y=y+x.x,y\in \mathbb R^n, x+y=y+x.x,y∈Rn,x+y=y+x..
  • Is associative. That is, for all vectors x,y,z∈Rn,(x+y)+z=x+(y+z)x,y,z\in \mathbb R^n, (x+y)+z=x+(y+z)x,y,z∈Rn,(x+y)+z=x+(y+z).
  • Has the zero vector as an identity. For all vectors x∈Rn,x+0=0+x=xx \in \mathbb R^n, x+\mathbf 0=\mathbf0+x=xx∈Rn,x+0=0+x=x where 0 is the vector of size n with 0 for each component.
  • Has an inverse, −x. That is x+(−x)=0x+(-x)=\mathbf 0x+(−x)=0.

The dot product of vectors

  • Is commutative. That is, for all vectors x,y∈Rn,xTy=yTxx,y\in R^n,x^Ty = y^Txx,y∈Rn,xTy=yTx.
