Other Norms

  • A norm is a function, in our case of a vector in Rn\mathbb{R}^nRn, that maps every vector to a nonnegative real number. The simplest example is the absolute value of a real number: Given α∈R\alpha \in \mathbb{R}α∈R, the absolute value of α, often written as |α|, equals the magnitude of α: $$\lvert \alpha \rvert = \left{
    \begin{array}{rl}
    \alpha & \text{if } \alpha \ge 0,\
    -\alpha & \text{otherwise}.
    \end{array} \right.$$
  • Similarly, one can find functions, called norms, that measure the magnitude of vectors. One example is the (Euclidean) length of a vector, which we call the 2-norm: for x∈Rnx \in \mathbb{R}^nx∈Rn, $$\lVert x \rVert 2 = \sqrt{\sum{i=0}{n-1}x_i2}$$
  • Other norms:
    • 1-norm (also called taxi-cab norm): $$\lVert x \rVert 1 = \sqrt{\sum{i=0}^{n-1}|x_i|}$$
    • For 1≤p≤∞1 \le p \le \infty1≤p≤∞, the p-norm: $$\lVert x \rVert p = \sqrt[p]{\sum{i=0}{n-1}|x_i|p}$$

  • Is associative. That is, for all vectors x,y,z∈Rn,(x+y)+z=x+(y+z)x,y,z\in \mathbb R^n, (x+y)+z=x+(y+z)x,y,z∈Rn,(x+y)+z=x+(y+z).
