Some terminology

  • row-echelon form:
    • The boxed values are known as the pivots.
    • In each row to the left of the vertical bar, the left-most nonzero element is the pivot for that row.
    • Notice that the pivots in later rows appear to the right of the pivots in earlier rows.
  • reduced row-echelon form:

Summary

  • Whether a linear system of equations Ax=bAx = bAx=b has a unique solution, no solution, or multiple solutions can be determined by writing the system as an appended system $$\left(A | b\right)$$ and transforming this appended system to row echelon form, swapping rows if necessary.
