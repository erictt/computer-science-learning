Matrix-Vector Multiplication with Special Matrices

  • When doing calculation, we always concern flops and memops. And later we will prove that, memops is much slower that flops.
  • Let’s take an example. If we want calculate y:=ATx+yy := A^T x + yy:=ATx+y. Normally, we will do, set B=ATB = A^TB=AT, then y:=Bx+yy := B x +yy:=Bx+y
    • Which means, we are going to spend the most time to transpose matrix A to B, and little time computing.
    • What we want is, compute with ATA^TAT without actually transposing.
    • The solution is, we can simply use columns of A for the dot products in the dot product based algorithm for y:=Ax+yy := Ax + yy:=Ax+y.
