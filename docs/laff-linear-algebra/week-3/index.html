Cost of Matrix-Vector Multiplication

  • Consider y:=Ax+y , where A∈Rm×ny := Ax+y\ \text{, where } A \in R^{m \times n}y:=Ax+y , where A∈Rm×n :
    • Notice that there is a multiply and an add for every element of A.
    • Since A has m×n=mnm \times n = mnm×n=mn elements, y:=Ax+yy := Ax+yy:=Ax+y, requires mn multiplies and mn adds, for a total of 2mn ﬂoating point operations (ﬂops).
