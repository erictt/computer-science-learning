Linear Transformations

  • A linear transformation is a vector function that has the following two properties:
    • Transforming a scaled vector is the same as scaling the transformed vector: $$L(\alpha x) = \alpha L(x)$$
    • Transforming the sum of two vectors is the same as summing the two transformed vectors: $$L(x + y) = L(x) + L(y)$$
  • Lemma: L:Rn→RmL: \mathbb{R}^n \to \mathbb{R}^mL:Rn→Rm is a linear transformation if and only if(iff) for all u,v∈Rnu,v \in \mathbb{R}^nu,v∈Rn and α,β∈Rn\alpha, \beta \in \mathbb{R}^nα,β∈Rn $$L(\alpha u + \beta v) = \alpha L(u) + \beta L(v)$$
  • Lemma: Let v0,v1,…,vk−1∈Rnv_0, v_1, \ldots, v_{k-1} \in \mathbb{R}^nv0​,v1​,…,vk−1​∈Rn and let L:Rn→RmL: \mathbb{R}^n \to \mathbb{R}^mL:Rn→Rm be a linear transformation. Then

    • then one can conclude that the property holds for all integers k≥kbk \ge k_bk≥kb​ . Often kb=0k_b = 0kb​=0 or kb=1k_b = 1kb​=1.

      • prove that f(αx)=αf(x)f(\alpha x) = \alpha f(x)f(αx)=αf(x) or f(x+y)=f(x)+f(y)f(x + y) = f(x) + f(y)f(x+y)=f(x)+f(y) for all α,x,y\alpha, x, yα,x,y.
